# attiny10-asm-templates-01

ATtiny10（AVR）のアセンブリ言語（Assembly）による開発用テンプレートおよびサンプルコード集です。
Microchip Studio (旧 Atmel Studio) での利用を想定しています。

## 概要 (Overview)

このリポジトリは、ATtiny10向けのアセンブラコードのテンプレートを提供します。


## 環境 (Requirement)

* **IDE:** Microchip Studio for AVR and SAM Devices 7.0
* **Target Device:** ATtiny10

## 使い方 (Usage)

1.  このリポジトリをクローンまたはダウンロードします。
2.  Microchip Studio で `attiny10-asm-templates-01.atsln` (または `.asmproj`) を開きます。
3.  プロジェクト内の `.asm` ファイルを編集して開発を進めてください。
